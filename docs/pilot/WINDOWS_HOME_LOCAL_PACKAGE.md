# Quantum HOME_LOCAL для Windows

## Назначение

Пакет предназначен для локальной обработки отчётов Wildberries на одном компьютере Windows 10/11. Он работает автономно после загрузки установочного файла, не отправляет отчёты во внешние сервисы и не выполняет запись на маркетплейс.

## Область текущей версии

- Поддерживаемая площадка: **Wildberries**.
- Поддержка Ozon отложена до утверждения схемы и профиля данных.
- Общее ядро проекта остаётся независимым от маркетплейса.
- Запись на маркетплейсы отключена.
- Производственный выпуск не разрешён без финальных проверок на целевом компьютере и реальном разрешённом отчёте.

## Основной запуск

1. Сохраните установочный файл в обычную локальную папку вне OneDrive, Dropbox, Google Drive и других синхронизируемых каталогов.
2. Запустите `Quantum_WB_Offline_Setup.exe`.
3. Установщик проверит встроенный пакет по SHA-256, установит или восстановит Quantum в `%LOCALAPPDATA%\QuantumLocalProduction` и запустит мастер первого запуска.
4. Выберите разрешённый к обработке файл XLSX.
5. Введите `AUTHORIZE`, чтобы подтвердить законные полномочия на обработку файла.
6. Проверьте определённую схему и отчётный период, затем введите `REVIEWED`.
7. После успешной обработки откроется локальный Центр решений Quantum и папка результатов.

`AUTHORIZE` и `REVIEWED` являются фиксированными контрольными токенами и намеренно не переводятся.

## Защита данных

- Microsoft Defender остаётся включённым при обычном запуске.
- При недоступности Defender разрешена только ограниченная структурная проверка; активное содержимое и повреждённые архивы блокируются.
- Каждый файл пакета проверяется по `manifest.sha256.json` до изменения установленной среды.
- Папки `config`, `data` и `output` сохраняются при восстановлении и повторной установке.
- Себестоимость, налог и прочие расходы не подставляются автоматически.
- Полный финансовый расчёт блокируется до передачи проверенного `finance_request`.
- Реальные отчёты, конфигурации и результаты нельзя размещать в GitHub или облачно-синхронизируемых папках.

## Средства восстановления

- `START_QUANTUM.cmd` — основной запуск одной кнопкой.
- `INSTALL_HOME_LOCAL.cmd` — установка или восстановление среды.
- `CONFIGURE_HOME_LOCAL.cmd` — создание конфигурации.
- `IMPORT_XLSX.cmd` — ручной импорт с обязательными подтверждениями.

## Установка по умолчанию

`%LOCALAPPDATA%\QuantumLocalProduction`

При разрешении Windows создаётся ярлык **«Центр решений Quantum»** на рабочем столе.

## Ограничения выпуска

Техническая успешность CI и встроенного self-test не заменяет:

1. установку, перезапуск, восстановление и откат на целевом компьютере пользователя;
2. проверку на реальном разрешённом отчёте Wildberries и финансовую сверку;
3. подпись Authenticode, когда требуется доверенный издатель Windows.
