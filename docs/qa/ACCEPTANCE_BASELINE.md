# Acceptance Baseline

## Ingestion

- Contract tests.
- Fuzz tests.
- Idempotency tests.
- Schema-drift tests.

## Finance

- Golden tests.
- Property-based tests.
- Differential tests.
- Reconciliation tests.
- Independent oracle.

## UI

- Critical-path end-to-end tests.
- Accessibility tests.
- Typed-state snapshots.

The actor changing a financial calculation must not approve the corresponding golden baseline.
