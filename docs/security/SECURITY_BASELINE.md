# Security Baseline

- Read-only marketplace integration for MVP.
- Network deny by default for agents where supported.
- No production credentials for Implementation Agent.
- Secret manager for credentials.
- Short-lived credentials.
- Tenant isolation.
- Input file limits and sandboxing.
- CSP, CSRF, XSS, and SQL injection controls.
- Privacy-safe logs.
- Dependency, vulnerability, license, SBOM, and provenance checks.
- GitHub Actions must be pinned by immutable commit SHA before activation.
