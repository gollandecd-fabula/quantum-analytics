# Exclusions and Security Boundaries

This checkpoint must not contain:

- access tokens, passwords, cookies, private keys, or signing secrets;
- Wildberries, Ozon, or other marketplace credentials;
- production database dumps;
- real or anonymized commercial reports;
- direct personal identifiers or pilot-user personal data;
- production environment values;
- ACTIVE commercial rules or profiles;
- Source Authority activation;
- marketplace write implementation;
- deployment authorization.

Repository visibility does not change these exclusions. Any discovered secret or personal data must be treated as a security incident, removed from history, and handled outside this checkpoint.
